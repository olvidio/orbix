# E43

Vista en pantalla del formulario E43 (certificado de calificaciones) de un alumno en una actividad:
datos personales, actividad y tabla de asignaturas matriculadas con nota, fecha y número de acta.
Incluye enlace para descargar el PDF.

## Manual De Usuario

Se abre con `id_pau` (actividad) e `id_nom` (persona) desde la navegación de dossiers. El
controller carga `e43_data` y muestra:

- Cabecera con DL destino/origen.
- Datos del alumno (nombre, nacimiento, fecha y lugar del seminario/CA/CV).
- Tabla de asignaturas matriculadas con calificación, fecha del acta y nº de acta.

El enlace **PDF** abre `e43_2_mpdf.php` en ventana nueva (parámetros firmados con `HashFront`).
Si hay error de datos, se muestra `msg_err` antes del contenido.
