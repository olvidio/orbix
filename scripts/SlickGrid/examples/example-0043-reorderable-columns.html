Demonstrates:

  • Reorderable columns

Instructions:

Drag the columns to reorder them. Uses jQuery UI's Sortable plugin behind the scenes. The first and last columns are disabled from reordering.

Usage:

Enable reorder on the grid. (Defaults to enabled.) 

var options = { enableColumnReorder = true };
grid = new Slick.Grid("#myGrid", data, columns, options);

You can disable reordering on a column-by-column basis with the reorderable column definition option. (Defaults to enabled.)

column.reorderable = true;
