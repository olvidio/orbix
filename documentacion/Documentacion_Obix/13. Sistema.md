# 🛠️ Grupo 13: Administración y Configuración

Este grupo reúne la administración de menús, DB y devel, permisos, procesos, usuarios web, cambios y configuración general.

> Nota: cada entrada del menú tiene una URL (punto de entrada). Los posibles pasos posteriores se describen en el **mapa** correspondiente dentro de `documentacion/Documentación Obix/`.

---

## 1) Menús

### 1.1 Gestor de menús
- **Entrada (menú):** `frontend/menus/controller/menus_que.php`
- **Mapa:** [[menus/mapa_menus_que]]
- **Acciones según mapa:**
  - Vista `menus_que.phtml` → `menus_get.php` → `menus_get.phtml`.
  - Desde `menus_get.phtml`: `menu_guardar.php`, `menu_eliminar.php`, `menu_mover.php`, `menu_copiar.php`.

### 1.2 Importar menús
- **Entrada (menú):** `frontend/menus/controller/menus_importar_form.php`
- **Mapa:** [[menus/mapa_menus_importar_form]]
- **Acciones según mapa:**
  - Vista `menus_importar_form.phtml` → `menus_importar.php`.

### 1.3 Exportar menús
- **Entrada (menú):** `frontend/menus/controller/menus_exportar_form.php`
- **Mapa:** [[menus/mapa_menus_exportar_form]]
- **Acciones según mapa:**
  - Vista `menus_exportar_form.phtml` → `menus_exportar.php`.

### 1.4 Menús a/desde ficheros
- **Exportar:** `src/menus/frontend/controller/menus_ficheros.php`  
  **Parámetros:** `accion=exportar`
- **Importar:** `src/menus/frontend/controller/menus_ficheros.php`  
  **Parámetros:** `accion=importar`
- **Mapa:** *(no existe `mapa_menus_ficheros` en la carpeta de documentación)*.

### 1.5 Meta menús (tabla)
- **Entrada (menú):** `frontend/shared/controller/tablaDB_lista_ver.php`  
  **Parámetros:** `clase_info=src\menus\domain\InfoMetaMenus`
- **Mapa:** [[shared/mapa_tablaDB_lista_ver]]

### 1.6 Ayuda de menús
- **Entrada (menú):** `frontend/menus/view/como.html`
